🏢 Real Estate Sales Analytics Dashboard

## 📌 Project Overview
End-to-end Business Intelligence solution built using:

- MySQL (Data Warehouse)
- Power BI (Data Modeling & Visualization)
- DAX (Advanced KPI Calculations)

## 🎯 Business Objectives
- Analyze booking conversion trends
- Identify top-performing builders
- Track quarterly revenue growth
- Evaluate sales channel efficiency
- Measure amenity impact on conversion

## 🏗 Data Model
Star Schema:

Dim_Builder
Dim_Market
Dim_Date
      ↓
fact_properties

## 📊 Key KPIs
- Total Revenue
- Booking Conversion Rate
- Avg Ticket Size
- QoQ Growth %
- YoY Growth %

## 🧠 Advanced Features
- Time Intelligence (QoQ, YoY)
- Ranking (Top 5 Builders)
- Market Share %
- Scatter Plot Correlation
- Sentiment Text Analysis

## 🛠 Tools Used
- MySQL
- Power BI
- DAX
- SQL

## 🚀 How to Use
1. Import schema.sql into MySQL
2. Load sample_data.csv
3. Open RealEstate_Dashboard.pbix
4. Refresh data source
